# Math-magician

> This is a react app that works as a calculator.


## Built With

- HTML
- CSS 
- Javascript
- React

## Getting Started

To get a local copy up and running follow these simple example steps.

- Clone repository in your local machine 
- cd Math-magician
- npm install
- npm start
